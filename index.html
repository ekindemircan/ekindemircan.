Two of the most recent games I've worked on. One was a redesign of an existing game, with new features added on top; the other was a brand-new game, designed from scratch.

Project 01

Find — Hidden Object

A full UI redesign of a live hidden-object game. The original interface worked but felt flat, so I rebuilt the visual language — every element redrawn, a real background added, and the whole thing polished while keeping it familiar to existing players. Alongside the refresh, I designed systems the game was missing: events, a season mode, and a richer shop.

RoleUI/UX Design
TypeRedesign + new features
PlatformMobile (iOS / Android)
FocusVisual system, live-ops, monetization
01
The main menu

The original home was a logo on an empty gradient. The redesign gives the game a place to live — a real background, the level progress brought up front, and events tucked into side buttons. It also branches into two states: the everyday menu, and a calmer Focus mode for the endless play that begins once the main levels are done.

Project 02

Jigsaw Puzzle

A jigsaw puzzle game built around a simple, fluid UI that keeps the level artwork front and center. The goal was a calmer, more neutral look — so the palette steps back and lets the images lead. The real challenge was designing clear, legible icons for the game's new gameplay features.

RoleUI/UX Design
TypeNew project · mode exploration
PlatformMobile (iOS / Android)
FocusSimple UI, neutral palette, gameplay icons
01
Main menu — three versions

The home screen sets the tone of a puzzle game — a calm, one-at-a-time ritual, or a library to browse and collect. To compare the feel of each, I designed three navigation versions side by side.

Version 01
Linear flow
The simplest path: just the current level's image, centered, with one Play button. No choice paralysis — you open the app and continue. Quiet and focused.
Version 02
Collection view
Puzzles are bought and played as themed collections — Special Days, animals, cars. Built around discovery and a sense of "completing the set."
Version 03
Mixed view
A hybrid: collection bands and single standalone puzzles share one feed. The richest option — browse sets and one-off images side by side.
The linear flow, in motion
The same minimal home doubles as a progress view: once you're inside a puzzle, it shows a live preview with a completion percentage, so the single image on the menu carries real state instead of being purely decorative.

02
Gameplay — light & dark

The gameplay screen comes in light and dark themes, each tuned for its setting rather than just recoloured. The board, piece tray, and every control adapt while staying legible.

Light theme
Dark theme
Custom gameplay tools
Beyond theming, I designed a set of gameplay-specific icons for the bottom bar — controls unique to a jigsaw game, like area selection and colour sorting, alongside edge-piece filters and a shuffle. Each had to read instantly at small sizes and work in both themes.
