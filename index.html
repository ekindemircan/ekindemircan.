Project 01

Find — Hidden Object

A full UI redesign of a live hidden-object game. The original interface worked but felt flat, so I rebuilt the visual language — every element redrawn, a real background added, and the whole thing polished while keeping it familiar to existing players. Alongside the refresh, I designed systems the game was missing: events, a season mode, and a richer shop.

RoleUI/UX Design
TypeRedesign + new features
PlatformMobile (iOS / Android)
FocusVisual system, live-ops, monetization
01
The main menu

The original home was a logo on an empty gradient. The redesign gives the game a place to live — a real background, the level progress brought up front, and events tucked into side buttons. It also branches into two states: the everyday menu, and a calmer Focus mode for the endless play that begins once the main levels are done.

03
Level end

The win screen was redesigned around a new like and share option. The completed scene becomes something players can react to and pass on, so finishing a level opens out instead of ending flat.

04
Season

A themed season system with collectible stickers. The onboarding is paced like a story — tappable, visual slides — and collected stickers fill a sticker book built to match the game's existing art. Players move a season scene forward by completing levels.

05
Events

A set of live events, each given its own colour so it reads at a glance — Combo Craze in gold, Streak Rise in orange, Victory Race in purple. The popup structure stays shared underneath; only the personality changes. Each has an entry screen and a results screen.

Project 02

Jigsaw Puzzle

A jigsaw puzzle game built around a simple, fluid UI that keeps the level artwork front and center. The goal was a calmer, more neutral look — so the palette steps back and lets the images lead. The real challenge was designing clear, legible icons for the game's new gameplay features.

RoleUI/UX Design
TypeNew project · mode exploration
PlatformMobile (iOS / Android)
FocusSimple UI, neutral palette, gameplay icons
01
Main menu — three versions

Three different takes on the home screen. The linear view is the simplest: one image, one Play button — calm and focused, with a live progress state once you're mid-puzzle. The collection view groups puzzles into themed sets for players who like to browse and complete. The mixed view combines both — collection bands and standalone images sharing a single feed.

01 · Linear view
02 · Collection view
03 · Mixed view
02
Gameplay — light & dark

The gameplay screen comes in light and dark themes, each tuned for its setting rather than just recoloured. The board, piece tray, and every control adapt while staying legible.

Light theme
Dark theme
03
Icons

A set of gameplay-specific icons for the bottom bar — controls unique to a jigsaw game, like area selection and colour sorting, alongside edge-piece filters and a shuffle. Each icon was designed to read instantly at small sizes and works across both light and dark themes.
