Hi, I'm Ekin!

I started my journey in architecture, where I learned how people move through spaces and how design can shape their experience. This made me curious about user behavior and how thoughtful design can guide people in a natural way.

Over time, I became especially interested in digital products and interactive experiences. Games first drew my attention because of how engaging and intuitive they can be, and that curiosity eventually led me to UX design.

Today, I design user experiences for mobile products. I enjoy creating interfaces that feel simple, clear, and enjoyable to use. My background helps me think about structure and flow while keeping the user's needs in mind. I like turning complex ideas into experiences that feel easy and natural for people.

Selected Work
Mobile Game UI/UX Design

Below is a selection of UI screens from different mobile games I've worked on. Each project has its own visual language and player expectations — the goal across all of them is the same: interfaces that feel consistent, intuitive, and tailored to the game's identity.

01 — Main Menu
Main Menu — Jigsaw Puzzle
Two navigation flows explored for the main menu of a jigsaw puzzle game. Both designs stay true to the game's calm, focused aesthetic while making core actions immediately accessible.
02 — Shop
Shop & Subscription — Jigsaw Puzzle
A calm, neutral shop design that stays consistent with the game's overall visual tone. The subscription screen intentionally breaks from that restraint — a richer, more royal purple signals exclusivity and makes the VIP Club feel like a premium tier apart from standard purchases.
03 — Achievements
Progress & Badge System
A scrollable list layout that prioritises readability over decoration. Each achievement is easy to scan at a glance — progress bars, reward indicators, and badge tiers are all clearly visible. Claimable rewards use a high-contrast green CTA to stand out immediately.
04 — Settings
Preferences & Accessibility
A monochrome, familiar interface format with no visual noise. Users can quickly scan and find what they need.
Popup System

These four screens share the same base popup design language — consistent components, layout structure, and visual tone. Each adapts slightly for its function, but the underlying system ensures players always feel at home, no matter which popup they encounter.

End State
Event Fail Screen
Shown at the end of a failed event run. Displays at-risk milestone rewards with a rewarded ad recovery CTA and a restart fallback.
Event
Event Start Popup
Shown when a player enters an active event. Features a countdown timer, event icon, and a brief description before the action begins.
Offer
No Ads Offer
A conversion-focused IAP popup with a bold visual, clear value proposition, and a single green CTA. Minimal friction, maximum clarity.
Profile
Edit Profile Popup
Username and avatar are editable from a single popup. Both actions in one place, reducing steps. Clear active state in the avatar grid.
Special Screens

Tutorial
Streak Master Onboarding

Level End
Like & Share

Leaderboard

The event leaderboard serves three purposes at once: players can track their own ranking, see how they compare against others, and claim their event rewards — all from a single screen at event end.

Event End

Podium layout for top 3 with a scrollable ranked list below. The player's own position is pinned at the bottom in green for immediate self-reference. Reward chests are shown per rank, making the stakes clear at a glance.
